## 📊 技术指标实现细节

### 1. MACD（指数平滑异同移动平均线）
```javascript
EMA_fast(12) - EMA_slow(26) = DIF
EMA(DIF, 9) = DEA
MACD = 2 * (DIF - DEA)
```
**特色**:
- 红绿柱区分多空力量
- DIF/DEA金叉死叉自动标注（预留功能）

### 2. KDJ（随机指标）
```javascript
RSV = (收盘 - 最低) / (最高 - 最低) * 100
K = SMA(RSV, 3)
D = SMA(K, 3)
J = 3K - 2D
```
**特色**:
- 超买超卖区域标注（>80超买，<20超卖）
- K/D/J三线颜色区分（蓝/橙/粉）

### 3. RSI（相对强弱指标）
```javascript
RS = 平均涨幅 / 平均跌幅
RSI = 100 - (100 / (1 + RS))
```
**特色**:
- 渐变填充效果
- 超买超卖虚线标注（70/30）

### 4. 布林带（Bollinger Bands）
```javascript
中轨 = SMA(Close, 20)
上轨 = 中轨 + 2 * StdDev
下轨 = 中轨 - 2 * StdDev
```
**特色**:
- 上中下轨颜色区分（红/橙/绿）
- 收盘价曲线叠加显示

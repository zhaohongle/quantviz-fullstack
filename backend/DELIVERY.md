## ⚠️ 注意事项

### 1. 端口冲突
如果端口 3001 被占用，使用：
```bash
PORT=3002 node server.js
```

### 2. 东方财富 API 不稳定
- 板块数据偶尔会失败（`socket hang up`）
- 系统会自动降级到股票数据聚合
- 不影响核心功能

### 3. 财务数据简化
- PE、PB、市值等字段使用估算值
- 真实生产环境建议集成 Wind / Tushare

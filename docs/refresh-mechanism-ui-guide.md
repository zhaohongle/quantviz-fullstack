## 🎬 交互动画时序

### 手动刷新流程

```
用户点击"刷新"按钮
      ↓
状态变为"刷新中"（蓝色脉动）
      ↓
刷新图标开始旋转
      ↓
调用 onRefresh 回调
      ↓
数据加载中...（0.5-2秒）
      ↓
加载成功
      ↓
状态变为"刷新成功"（绿色）
      ↓
最后更新时间变为"刚刚"
      ↓
等待 2 秒
      ↓
状态恢复为"就绪"（灰色）
```

### 自动刷新流程

```
页面加载完成
      ↓
RefreshManager.init() 初始化
      ↓
如果 autoStart = true，启动定时器
      ↓
等待 60 秒（默认间隔）
      ↓
自动触发刷新（与手动刷新流程相同）
      ↓
等待下一个间隔
      ↓
循环...
```

## 🔍 关键代码片段

### 刷新按钮禁用状态
```css
.refresh-btn:disabled {
    opacity: 0.6;
    cursor: not-allowed;
}
```

### 旋转动画
```css
@keyframes spin {
    from { transform: rotate(0deg); }
    to { transform: rotate(360deg); }
}

.refresh-icon.spinning {
    animation: spin 1s linear infinite;
}
```

### 脉动动画
```css
@keyframes pulse {
    0%, 100% { opacity: 1; }
    50% { opacity: 0.5; }
}

.status-indicator.loading {
    animation: pulse 1.5s ease-in-out infinite;
}
```

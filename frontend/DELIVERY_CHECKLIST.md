## 🔄 回滚方案

如果需要回滚到浅色主题：

### 方法 1: 使用备份文件
```bash
cd quantviz-fullstack/frontend/pages
for file in $(find . -name "*.backup"); do
  original="${file%.backup}"
  cp "$file" "$original"
  echo "已回滚: $original"
done
```

### 方法 2: Git 回滚（如果使用版本控制）
```bash
cd quantviz-fullstack
git checkout HEAD -- frontend/pages/
```

### 方法 3: 手动移除主题引用
删除每个页面中的：
```html
<link rel="stylesheet" href="../../css/dark-theme.css">
```

## 📚 维护指南

### 新增页面时

1. **添加主题引用**
   ```html
   <head>
     ...
     <link rel="stylesheet" href="../../css/dark-theme.css">
   </head>
   ```

2. **使用 CSS 变量**
   ```css
   body {
     background: var(--bg);
     color: var(--text);
   }
   
   .card {
     background: var(--bg-card);
     border: 1px solid var(--border);
   }
   ```

3. **参考现有页面**
   - 推荐详情页（完整示例）
   - AI 推荐页（列表布局）

### 更新主题色时

编辑 `css/dark-theme.css`：
```css
:root {
  --primary: #00d4ff;  /* 修改青色主题 */
  --up: #00e676;       /* 修改涨色 */
  --down: #ff5252;     /* 修改跌色 */
}
```

所有页面自动生效！

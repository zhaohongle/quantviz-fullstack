## ⚠️ 已知问题

### 需要验证的问题

1. **页面内容是否完整**
   - 部分页面可能需要适配（移除独立导航）
   - 样式路径可能需要调整

2. **脚本依赖**
   - 页面内的脚本是否正确执行
   - 是否有跨域问题

3. **数据加载**
   - API 请求是否正常
   - Mock 数据是否可用

### 解决方案

如遇问题，按以下步骤排查：

1. **打开浏览器开发者工具**
   - F12 或 Cmd+Opt+I

2. **查看 Console 面板**
   - 检查 JavaScript 错误

3. **查看 Network 面板**
   - 检查请求是否成功（200 状态码）
   - 检查资源加载情况

4. **查看 Elements 面板**
   - 检查 DOM 结构是否正确
   - 检查样式是否生效

## 🚀 快速部署

### 方法1：本地预览

```bash
cd /Users/qihoo/Desktop/openclaw/quantviz-updated
python3 -m http.server 8000
# 访问 http://localhost:8000
```

### 方法2：Vercel部署（5分钟）

1. 访问 https://vercel.com
2. 登录GitHub账号
3. 点击 "New Project"
4. 上传 `quantviz-updated` 文件夹
5. 点击 "Deploy"
6. 完成！

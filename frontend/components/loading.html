📐 加载/空/错误状态组件规范

1. 加载状态（Loading States）

加载中...
正在加载数据...
使用场景：
• 旋转圆圈：全局加载、按钮提交
• 骨架屏：列表/卡片首次加载
• 进度条：大文件上传、数据导入
<!-- 旋转圆圈 --> <div class="loading-spinner"> <div class="spinner"></div> <div class="loading-text">加载中...</div> </div> <!-- 骨架屏 --> <div class="skeleton-card"> <div class="skeleton-avatar"></div> <div class="skeleton-line" style="width: 80%;"></div> <div class="skeleton-line" style="width: 60%;"></div> </div> <!-- CSS 动画 --> @keyframes spin { to { transform: rotate(360deg); } } @keyframes skeleton-loading { 0% { background-position: 200% 0; } 100% { background-position: -200% 0; } }

2. 骨架屏 - 表格/列表

表格骨架屏：
• 模拟表格行列结构
• 使用不同宽度表示不同列
• 适合成分股列表、资金流向表格

3. 空状态（Empty States）

📭
暂无数据
当前没有可显示的内容
⭐
暂无自选股
快去添加你感兴趣的股票吧
🔍
未找到相关结果
请尝试其他关键词
使用场景：
• 无数据：列表为空
• 无自选股：用户未添加自选
• 无搜索结果：搜索无匹配项
<div class="empty-state"> <div class="empty-icon">📭</div> <div class="empty-title">暂无数据</div> <div class="empty-text">当前没有可显示的内容</div> <button class="btn-action">刷新页面</button> </div>

4. 错误状态（Error States）

⚠️
数据加载失败
网络连接异常，请重试
🔧
服务器错误
服务暂时不可用，请稍后再试
🔒
权限不足
此功能需要升级会员
使用场景：
• 加载失败：网络错误、API 调用失败
• 服务器错误：500/503 错误
• 权限不足：未登录或权限受限
<div class="error-state"> <div class="error-icon">⚠️</div> <div class="error-title">数据加载失败</div> <div class="error-text">网络连接异常，请重试</div> <button class="btn-retry">重新加载</button> </div>

5. 成功状态（Success States）

✅
添加成功
已添加到自选股
🎉
提交成功
您的反馈已收到，感谢支持
使用场景：
• 操作成功提示（短暂显示后自动消失）
• 表单提交成功
• 文件上传完成

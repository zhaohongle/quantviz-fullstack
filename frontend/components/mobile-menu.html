📱 移动端菜单组件规范

1. 移动端顶部导航栏
量子财富
设计要点：
• 高度：48px
• 背景色：#1E3A8A（深蓝）
• 汉堡图标：24px，居右
• 点击汉堡图标打开侧边栏
2. 侧边栏菜单（Drawer）
设计要点：
• 宽度：280px（最大 80vw）
• 从左侧滑入，动画时间 0.3s
• 包含用户信息、主菜单、底部功能
• 点击遮罩层或关闭按钮收回
• 支持滚动（内容过多时）
3. 底部标签栏（Tab Bar）
📊 指数 💰 资金流 🤖 AI推荐 ⭐ 自选
设计要点：
• 高度：56px
• 固定在底部（position: fixed）
• 4-5 个主要功能入口
• 图标 22px，文字 12px
• Active 状态：#1E3A8A
• 默认状态：#6B7280
4. JavaScript 交互代码

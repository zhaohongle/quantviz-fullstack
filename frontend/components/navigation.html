1. 桌面端导航栏（≥768px）

状态说明：
• 默认状态：白色文字，透明边框
• Hover 状态：透明度 0.8，底部橙色边框
• Active 状态：底部橙色边框
• 搜索框 Focus：背景变亮，橙色边框，宽度增加至 250px
2. 移动端导航栏（<768px）
🌊 量子财富
移动端菜单：
• 点击汉堡菜单图标打开侧边栏
• 侧边栏从左侧滑入，宽度 80%（最大 300px）
• 包含半透明遮罩层
• 点击遮罩或关闭按钮收回侧边栏

4. CSS 代码示例
/* 导航栏基础样式 */ .navbar { background-color: #1E3A8A; padding: 16px 32px; display: flex; justify-content: space-between; align-items: center; } /* 导航链接 */ .nav-link { color: #FFFFFF; font-size: 14px; font-weight: 500; padding: 8px 0; border-bottom: 2px solid transparent; transition: all 0.2s; } .nav-link:hover { opacity: 0.8; border-bottom-color: #F59E0B; } .nav-link.active { border-bottom-color: #F59E0B; } /* 搜索框 */ .search-input:focus { background-color: rgba(255,255,255,0.15); border-color: #F59E0B; width: 250px; } /* 响应式 */ @media (max-width: 768px) { .navbar-center, .navbar-right { display: none; } .mobile-menu-btn { display: block; } }
5. JavaScript 交互示例
// 用户菜单交互 document.querySelector('.user-avatar').addEventListener('click', function(e) { e.stopPropagation(); document.querySelector('.user-dropdown').classList.toggle('show'); }); document.addEventListener('click', function() { document.querySelector('.user-dropdown').classList.remove('show'); }); // 移动端菜单交互 document.querySelector('.mobile-menu-btn').addEventListener('click', function() { document.querySelector('.mobile-menu').classList.add('show'); document.querySelector('.menu-overlay').classList.add('show'); }); document.querySelector('.menu-close-btn').addEventListener('click', function() { document.querySelector('.mobile-menu').classList.remove('show'); document.querySelector('.menu-overlay').classList.remove('show'); }); document.querySelector('.menu-overlay').addEventListener('click', function() { document.querySelector('.mobile-menu').classList.remove('show'); this.classList.remove('show'); });

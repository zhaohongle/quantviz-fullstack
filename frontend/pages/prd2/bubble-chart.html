📊 板块资金流向气泡图
📊
气泡图视图（ECharts 集成）
X 轴：资金流向金额（流入为正，流出为负）
Y 轴：板块涨跌幅
气泡大小：成交额
气泡颜色：红色（流入）/ 绿色（流出）

开发说明：使用 ECharts 散点图（scatter）实现，详见下方代码示例
📌 图例说明
资金流入板块（红色气泡）
资金流出板块（绿色气泡）
小气泡：成交额较小
大气泡：成交额较大
💻 ECharts 实现代码
// 数据格式 const sectorData = [ { name: '半导体', amount: 28512, changePct: 4.23, volume: 124500 }, { name: '新能源', amount: 21233, changePct: 3.15, volume: 98200 }, { name: '银行', amount: -8545, changePct: -1.52, volume: 45600 }, // ... 更多板块数据 ]; // ECharts 配置 const option = { title: { text: '板块资金流向气泡图', left: 'center' }, tooltip: { trigger: 'item', formatter: function(params) { const data = params.data; return `${data[3]}
资金流向: ${data[0] > 0 ? '+' : ''}${data[0].toFixed(2)} 亿
涨跌幅: ${data[1] > 0 ? '+' : ''}${data[1].toFixed(2)}%
成交额: ${data[2].toFixed(2)} 亿`; } }, xAxis: { name: '资金流向（亿元）', nameLocation: 'center', nameGap: 30, splitLine: { show: true, lineStyle: { type: 'dashed', color: '#E5E7EB' } }, axisLine: { lineStyle: { color: '#6B7280' } } }, yAxis: { name: '涨跌幅（%）', nameLocation: 'center', nameGap: 40, splitLine: { show: true, lineStyle: { type: 'dashed', color: '#E5E7EB' } }, axisLine: { lineStyle: { color: '#6B7280' } } }, series: [ { name: '资金流入', type: 'scatter', symbolSize: function(data) { // 根据成交额计算气泡大小（10-80px） return Math.max(10, Math.min(80, data[2] / 2000)); }, data: sectorData .filter(d => d.amount > 0) .map(d => [d.amount / 100, d.changePct, d.volume / 100, d.name]), itemStyle: { color: '#EF4444', opacity: 0.7 }, label: { show: true, formatter: '{@[3]}', position: 'inside', fontSize: 11, color: '#FFFFFF' } }, { name: '资金流出', type: 'scatter', symbolSize: function(data) { return Math.max(10, Math.min(80, data[2] / 2000)); }, data: sectorData .filter(d => d.amount < 0) .map(d => [d.amount / 100, d.changePct, d.volume / 100, d.name]), itemStyle: { color: '#10B981', opacity: 0.7 }, label: { show: true, formatter: '{@[3]}', position: 'inside', fontSize: 11, color: '#FFFFFF' } } ] }; // 初始化图表 const chartDom = document.getElementById('chart-container'); const myChart = echarts.init(chartDom); myChart.setOption(option); // 响应式调整 window.addEventListener('resize', function() { myChart.resize(); }); // 点击事件 myChart.on('click', function(params) { const sectorName = params.data[3]; // 跳转到板块详情页 window.location.href = `/sector/${sectorName}`; });
📦 依赖库：
需要引入 ECharts 库：
<script src="https://cdn.jsdelivr.net/npm/echarts@5/dist/echarts.min.js"></script>

🎨 设计要点：
• X 轴正值为流入（右侧），负值为流出（左侧）
• Y 轴正值为上涨，负值为下跌
• 气泡大小与成交额成正比
• 流入用红色（#EF4444），流出用绿色（#10B981）
• 气泡内显示板块名称
• 点击气泡跳转到板块详情页

🔄 交互说明：
• 鼠标悬停显示详细数据（Tooltip）
• 点击气泡跳转到板块详情
• 支持缩放和拖拽（dataZoom）
• 响应式自适应
